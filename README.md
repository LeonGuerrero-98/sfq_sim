# sfq_sim

`sfq_sim` is a Python packaged for simulating Single-Flux-Quantum (SFQ) digital control logic applied to superconducting qubits.

## To install

1. Navigate to folder containing `sfq_sim`
2. Run `pip install .` in the terminal
